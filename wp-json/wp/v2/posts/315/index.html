    1). What are conversions & how to measure them?<\/strong><\/h2>\n

    A conversion in Adwords is the number of times your goal is being completed. For example, you are running an AdWords campaign to drive traffic to your website. Your landing page consists of a subscription form and it goes to a thank you page after the form is submitted.<\/p>\n

    Here, your goal is to drive people to your landing page using the ad, fill out the subscription form and redirect them to the thank you page. So, the number of people coming from your Adwords ads and subscribing to your newsletter are known as Adwords conversions.<\/p>\n

    There are two ways we can measure conversions in Adwords. One is to create a custom conversion and paste the conversion code to our final thank you page. The second way is to directly import Google Analytics goals into Adwords.<\/p>\n

    In this guide, we will talk about creating a custom conversion code and pasting it onto our conversion page.<\/p>\n

    2). Setup Adwords Conversion Tracking<\/strong><\/h2>\n

    Adwords provides a step by step process to set up conversions. If you are new to Adwords and this is your first-time conversion tracking, follow the simple steps below.<\/p>\n

    Step1:<\/strong> Log in to your Adwords account<\/p>\n

    Step2:<\/strong> Click on the settings icon on the top right section of the page, and under measurement<\/strong> click conversions<\/strong>.<\/p>\n

    <\/p>\n

    Step3: <\/strong>If this is your first conversion, click on the \u2018Conversion\u2019 button.<\/p>\n

    <\/p>\n

    Step4: <\/strong>Select the source of conversion as \u2018Website\u2019.<\/p>\n

    <\/p>\n

    Step5: <\/strong>Enter the conversion action details.<\/p>\n

      \n
    1. Name: Enter a specific name of conversion<\/li>\n
    2. Category: select the type of conversion you want to set up i.e Lead, signup, purchase or pageview<\/li>\n
    3. Value: Assign a value to your conversion. If you don\u2019t understand you can leave it empty<\/li>\n
    4. Count: choose how you wish to count the conversions. Select \u2018one\u2019 for counting only unique conversions and \u2018every\u2019 for counting all conversions.<\/li>\n
    5. Conversion windows: select for how long you want to count the visitor as a conversion. For example, if you set the conversion window as 30, a conversion will be counted when a visitor purchases within 30 days of his\/her visit.<\/li>\n
    6. Include in Conversions: check to include the conversion action in your \u2018Conversions\u2019 column. You can leave as it is if you don\u2019t understand it.<\/li>\n<\/ol>\n

      Attribution Model: select how you want to attribute the conversion to the ad. Learn more about attribution modeling here<\/a>.<\/p>\n

      Click on \u2018Create and Continue\u2019 button.<\/p>\n

      Step6: To set up conversion tracking, the new setup requires you to add the global site tag<\/a> on every page of your website. You will still be able to track conversion without the gTag. However, adding it on every page is now required. Copy the global site tag on all pages of your website if you haven\u2019t already added it.<\/p>\n

      Step7: <\/strong>next, choose when you want to fire the conversion tag. Select page load if you want the conversion to fire each time a page loads, and select event when you want it to fire upon a specific action is completed i.e button click.<\/p>\n

      3). Google Tag Manager Adwords Conversion Tracking<\/strong><\/h2>\n
      I suppose that you already have a Google Tag Manager account running and if not, you can type tagmanager.google.com<\/a> in your browser and follow the instructions to set up an account.<\/p>\n
      Follow the below steps to add Adwords conversion tracking tag via Google Tag Manager.<\/p>\n
      Note:<\/strong> For the sake of this article, let\u2019s assume that you are new to Google Tag Manager, so I can explain every single step in detail, and that is how we can get the maximum number of people to benefit from it.<\/p>\n
      Step1:<\/strong> log in to your tag manager account<\/p>\n
      Step2:<\/strong> create a new tag by clicking on the \u2018New Tag\u2019 tab.<\/p>\n
      <\/p>\n
      Step3:<\/strong> Enter the name of the tag as \u2018Adwords Lead Conversion Tag\u2019. Here, Lead is the conversion category I selected in the previous step. You can change this according to your conversion type so this can be uniquely identified among many other tags.<\/p>\n
      Step4:<\/strong> Click on Tag Configuration and select the tag type as \u2018Adwords Conversion Tracking\u2019.<\/p>\n
      <\/p>\n
      Step5:<\/strong> Go back to Adwords and copy the conversion ID and conversion Label I highlighted in the previous step. Paste those values in the appropriate fields in Adwords tag configuration.<\/p>\n
      <\/p>\n
      Note: you can also send a dynamic conversion value through GTM<\/a>.<\/p>\n
      Step6: <\/strong>Click on the advanced settings and select the tag firing option as once per page.<\/p>\n
      <\/p>\n
      Step7: <\/strong>Click on the \u2018Triggering\u2019 tab and create a new trigger to fire this tag on specific pages. Enter a unique name for the trigger. Select the trigger type as \u2018Page View\u2019, and choose the trigger to fire on some pages.<\/p>\n
      It\u2019s important to understand this step here. We have to set the condition so that it will only fire the conversion code to the thank-you page only, not on all pages.<\/p>\n

      <\/p>\n
      All the triggers would be the same except the page path. Look at your thank you page URL and paste the page path the in a specific field and save the trigger.<\/p>\n
      Step8:<\/strong> Save the tag.<\/p>\n
      <\/p>\n
      Step9: <\/strong>test and debug this tag before setting it live on the page. Click on the preview button on the main page.<\/p>\n
      <\/p>\n
      Step10: <\/strong>Open your thank you page in a new page to verify if the tag is firing correctly. If you have followed the steps carefully, you will see the Adwords tag firing in the preview tab.<\/p>\n
      <\/p>\n
      If the Adwords tag isn\u2019t firing, you might have missed a step. Go to step one and revise if you have followed all the steps properly.<\/p>\n
      Step11: Go back to your tag manager account, click on leave preview mode link, and click on the submit button to publish the changes.<\/p>\n
      4). Add Conversion Linker Tag<\/strong><\/h2>\n
      Now, with the new update, you have to add the conversion linker tag<\/a> also. To do this, create a new tag, select the tag type as conversion tag and set all pages trigger.<\/p>\n

      Finally, publish changes in your container.<\/p>\n
      5). Verify Adwords Conversion Tracking<\/strong><\/h2>\n
      To verify the conversion tag you have added on the page is correct or not, we can test it with Google Tag Assistant. To do this:<\/p>\n
