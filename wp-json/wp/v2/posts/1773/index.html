If you are looking to install the new LinkedIn insight tag via Google Tag Manager template, then this is for you.<\/p>\n

The LinkedIn insight tag is a javascript code snippet that is placed on your website for conversion tracking<\/a> and retargeting for your LinkedIn ad campaigns.<\/p>\n

The new Google Tag Manager<\/a> interface has introduced a new LinkedIn Insight template for a hassle free installation. You don\u2019t need to follow the conventional way of copying and pasting the code.<\/p>\n

LinkedIn Insight Tag Via Google Tag Manager<\/strong><\/h2>\n

Follow the steps below to install LinkedIn Insight tag via Google Tag Manager on your website.<\/p>\n

    \n
  1. Login to your google tag manager account and select the specific container where you want to add the LinkedIn insight.<\/li>\n
  2.  Inside the workspace, click on the New Tag button<\/li>\n<\/ol>\n

    <\/p>\n

    3. Enter the Tag Name as LinkedIn Insight<\/p>\n

    4. Click on the tag configuration and select the tag type as LinkedIn Insight<\/p>\n

    5. Enter your Partner ID in the empty field<\/p>\n

    <\/p>\n

    6. Click on the triggering button and select on which pages you want to fire this tag. To fire this tag on all pages, select the All Pages trigger.<\/p>\n

    <\/p>\n

    Your LinkedIn insight tag should look similar to the below image.<\/p>\n

    <\/p>\n

    7. Now, save this tag and click on the preview button and open your website in a new tag to test this tag. Publish changes in GTM once you test the new tag. <\/p>\n
