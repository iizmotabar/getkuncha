For marketers, creating a marketing campaign is of supreme importance, which takes ample amount of time, strategic thinking and market research. Let\u2019s assume you have invented a really creative and unique marketing campaign after putting your efforts. <\/span><\/p>\n

The next important step is to measure the success of your marketing campaign. This \u00a0requires the tools to analyze campaigns, generating reports, understanding the weak points and improving your marketing campaigns to make it even more successful.<\/span><\/p>\n

In this article, you will learn about the three important things to measure the success of your marketing campaigns. You would need to familiarize yourself with google analytics which will be your favorite companion. <\/span><\/p>\n

1. Determine the Source of your Audience<\/strong><\/h2>\n

For running a successful marketing campaign you have to discover the source of your traffic. How would you know which campaign is driving more traffic? Which marketing campaigns are driving the most successful leads? <\/span><\/p>\n

Let\u2019s move on to further understand in detail how you can actually capture the source of your campaigns in analytics account. <\/span><\/p>\n

Using UTM Parameters<\/strong><\/h3>\n

UTM(Urchin Tracking Module) is a simple code\/parameters you can add to the end of your custom URL in order to track the source of traffic. It is also known as UTM parameters which send data to your analytics account. <\/span><\/p>\n

You can track five things using UTM parameters.<\/span><\/p>\n

1. Campaign<\/strong><\/p>\n

It is the unique campaign name or promo code for a single campaign<\/span><\/p>\n

utm_campaign=\u201dwintersale20percent\u201d<\/span><\/p>\n

2. Source<\/strong><\/p>\n

The site that is responsible for sending you the traffic. It can be facebook, twitter, google+ or any other site. <\/span><\/p>\n

utm_source=\u201dtwitter\u201d<\/span><\/p>\n

3. Medium<\/strong><\/p>\n

The type of medium used for your campaign. For example social media, email or ads<\/span><\/p>\n

utm_medium=\u201demail\u201d<\/span><\/p>\n

4. Content<\/strong><\/p>\n

This parameter is used to differentiate between links and content within the same ad. If you have two identical links on the same page you can use utm_content to differentiate between them. <\/span>utm_source=\u201dfooterlink\u201d<\/span><\/p>\n

5. Term<\/strong><\/p>\n

This parameter is specifically used for paid keyword terms. If you are paying for <\/span>keywords using PPC, this must be used.<\/span><\/p>\n

utm_term=\u201demail+marketing\u201d<\/span><\/p>\n

For tracking any URL you must use the first three parameters(campaign, source, medium) to get the necessary data in analytics. Below is an example of how UTM parameters are used:<\/span><\/p>\n

https:\/\/marketlytics.com\/blog\/free-content-creation-and-marketing-tools<\/span>?<\/span>utm_medium=social<\/span>&<\/span>utm_campaign=blog<\/span>&<\/span>utm_source=twitter<\/span><\/a><\/p>\n

Let\u2019s break down the above URL:<\/span><\/p>\n

URL:<\/strong> https:\/\/marketlytics.com\/blog\/free-content-creation-and-marketing-tools<\/span><\/a><\/p>\n

Utm_medium:<\/strong> social<\/span><\/p>\n

Utm_campaign:<\/strong> blog<\/span><\/p>\n

Utm_source<\/strong>: twitter<\/span><\/p>\n

There is no any such rule defined for using UTM parameters. You can use in any order but make sure to separate each with \u2018&\u2019 and add \u2018?\u2019 after the end of URL.<\/span><\/p>\n

If you are not familiar with using UTM parameters, below are a few tools that make adding these parameters easy:<\/span><\/p>\n

utm.io<\/a><\/strong><\/p>\n

google url builder<\/p>\n

Google campaign URL builder is also a very simple tool for adding UTM parameters. You just need to enter the required information and it will automatically build URL for you.<\/span><\/p>\n

How to see campaign reports in Analytics?<\/strong><\/h3>\n

Once you added UTM parameters to your campaigns, you can easily see data in google analytics. In order to view campaign reports, navigate to <\/span>Acquisition >> Campaign >> All Campaigns<\/strong>.<\/span><\/p>\n

all campaign reports<\/p>\n

The above reports show data about different campaigns, you can change the primary dimensions to view data about the only campaign, source and medium. You can measure which marketing campaign is successful and which is more relevant to your audience. <\/span><\/p>\n

2. Consistent Codes on All Pages<\/strong><\/h2>\n

Google analytics will only track the pages where you have inserted the analytics tracking code. In order not to miss any data, you must add analytics tracking code to <\/span>all pages<\/strong> of your website. <\/span><\/p>\n

If in any case you have missed adding <\/span>analytics tracking code<\/span><\/a> on a page, you would not be able to see accurate reports. If you are not viewing accurate reports then you are not able to measure the accuracy of your marketing campaigns. \u00a0<\/span><\/p>\n

Indications for Missing Tracking Code<\/strong><\/h3>\n

How do you know if the tracking code is missing on a specific page? Below are some indications for missing tracking code.<\/span><\/p>\n

    \n
  • If the tracking code is missing from your site, you will not be able to see real time and any other reports.<\/li>\n
  • If the tracking code is missing on a single page, that page reports will be missing from the overall reports.<\/li>\n<\/ul>\n
