      Segment and analyze your audience based on similar activities, events and behavior.<\/p>\n<\/li>\n

      \n Setting up Google Analytics
      \n<\/h2>\n

      To install google analytics on your website, you first handedly need a google account. You can also use a pre-existing account (if you have one) otherwise you will need to register for a new one.<\/p>\n

      Integrating and directing, all your website\u2019s classified data to an external tool, requires a certain level of security and indemnity for you to trust it.<\/p>\n

      To achieve that, you must ensure your google account is secure and only you have access permissions because the last thing you want is anyone compromising your data or gaining illegal access to it.<\/p>\n

      \u00a0<\/h3>\n

      \n Create a Google Analytics account
      \n<\/h2>\n

      If you have your google account you do not need to sign up for a google analytics account. You can visit https:\/\/google.com\/analytics<\/a> and sign in with your pre-existing account. The next window will display three steps to setup Google Analytics.<\/p>\n

      <\/p>\n

      Click on the sign up button to get started with the implementation process and fill in the required information for your website.<\/p>\n

      <\/p>\n

      Once you populated the form with all necessary credentials, click the get tracking ID button.
      The next important thing is to copy and paste the provided tracking code snippet into your website.<\/p>\n

      <\/p>\n

      \n\tAdding google analytics to any website
      \n<\/h2>\n

      Copy the code between the script tags from the last step and open your website admin panel. Paste this code just before the closing head tag(<\/head>) on every page of your website.<\/p>\n

      \n How to Install Google Analytics on Wordpress
      \n<\/h2>\n

      Wordpress is used by 27.3% of all the websites, that is a Content Management System(CMS) market share of 58.8%. The reason CMS is immensely popular amongst marketers is because of its simplicity, ease of installation and user-friendliness.<\/p>\n

      Source: websitesetup.org<\/p><\/div>\n

      Wordpress.com or Wordpress.org<\/strong><\/h3>\n

      Above all things, you should understand the difference between wordpress.com & wordpress.org. Wordpress.com is free to avail for anyone who is willing to create an online presence for its business or blog, hosted on the Wordpress server. Whereas, wordpress.org allows you to download the Wordpress setup and install on your own server. Keep in mind, you can not install Google Analytics on any wordpress.com blog.<\/p>\n

      There are two best methods to install google analytics in wordpress. You can either add analytics tracking code to header.php file or install google analytics plugin by Monsterinsights.<\/p>\n

      If you are familiar with coding I would suggest you add manually otherwise skip to second method.<\/p>\n

      \u00a0<\/p>\n

      Method1: Install Google Analytics in Wordpress Manually<\/strong><\/h3>\n

      I am assuming that you have your analytics tracking code and if you haven\u2019t yet, please click here<\/a>\u00a0to start with analytics and get the\u00a0tracking code. Once you get the\u00a0tracking code, follow the steps below:<\/p>\n

      1. Login to your WordPress dashboard and go to Editor from the left navigation menu.<\/p>\n

      <\/p>\n

      2. From the list of templates on the right side, click on header.php<\/strong> file. If you can\u2019t find the file, press Ctrl+F and search for header.php.<\/p>\n

      <\/p>\n

      3. Inside the header.php file, look for the tag <\/head>.<\/strong> Copy your analytics tracking code and paste it just above the <\/head> tag.<\/p>\n

      <\/p>\n

      4. Click on the update file button.<\/p>\n

      To verify if you set up the tracking code correctly, scroll down to Verify your analytics tracking id topic. <\/p>\n

      Method2: Google Analytics plugin by Monsterinsights<\/strong><\/h3>\n

      Google analytics plugin by monsterinsights is the most popular analytics wordpress plugin. It is active on more than 1 million wordpress websites. To install this plugin follow the steps below:<\/p>\n

      1. Login to your wordpress dashboard and click on ‘add new’ plugin from the left navigation menu.<\/p>\n

      <\/p>\n

      2. In the search bar enter \u2018google analytics monsterinsights\u2019.<\/p>\n

      3. Click on the install now button on the Google Analytics for WordPress by MonsterInsights<\/p>\n

      4. Once you install and activate the plugin, a new item will appear named Insights in the left navigation menu. Click on Insights >> Settings to add your analytics account.<\/p>\n

      <\/p>\n

      5. Inside settings click on the \u201cAuthenticate with your google account\u201d button.<\/p>\n

      <\/p>\n

      6. Follow the steps ahead to connect your analytics account.<\/p>\n

      Once the connection is successful, follow the steps below to verify setup.<\/p>\n

        \n Verify your analytics tracking ID
        \n<\/h2>\n

        There are several ways to verify if you have properly setup your analytics account. The simplest one is to open your website in a new tab and check real time reports in Google Analytics. You can find real-time reports in the left menu I.e Real-Time>> Overview. If your installation is correct, you will see the number of active visitors on your site at that very moment.<\/p>\n

        <\/p>\n

        If you don\u2019t see any real time visitors, then there might be a problem with your tracking code. To cross-check your analytics tracking id, install the Google<\/a> Tag<\/a> Assistant<\/a> browser extension.
        Once you get this extension, open your website in a new tab, click on the tag assistant icon, click the enable button and then reload the page.<\/p>\n

        <\/p>\n

        If there is an error in your analytics tracking code snippet, this window will display the error with message and the tag assistant icon color will be red. The green icon means your tracking code is good to go.<\/p>\n

        <\/p>\n

        Note: If the setup is correct, the data in your google analytics account will appear within the next 24-48 hours.<\/strong><\/p>\n
