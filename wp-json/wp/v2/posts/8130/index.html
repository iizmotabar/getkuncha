There are two methods to import historic data to Mixpanel. The first one is via \/import API<\/a> and the other way is by using the tool Segment CSV importer.<\/a>  The most easier way you can do is by using the tool.<\/p>\n

The tool is built on top of segment.com<\/a> and you can use it to create a CSV of any data you want to upload and send it one go. If you are already using segment, the setup is even faster.<\/p>\n

To import historic data into Mixpanel you should have the data in CSV file and the write key of your segment account. Follow the steps below to import data:<\/p>\n

Note: <\/strong>Before uploading your CSV file make sure to enable Mixpanel Integration<\/a> in your Segment account.<\/p>\n

Step#1: Visit Segment CSV Importer<\/a><\/p>\n

Here you would need to have two things:<\/p>\n

    \n
  1. \n

    The write key(find inside your segment account)<\/p>\n<\/li>\n

  2. \n

    CSV file<\/p>\n<\/li>\n<\/ol>\n

    <\/p>\n

    Step#2:<\/strong> Enter the write key into the write field<\/p>\n

    Step#3:<\/strong> Click on the choose button to upload your CSV file containing data<\/p>\n

    Your CSV file should have the following fields:<\/p>\n

      \n
    1. \n

      Action<\/p>\n<\/li>\n

    2. \n

      User ID<\/p>\n<\/li>\n

    3. \n

      Anonymous ID<\/p>\n<\/li>\n

    4. \n

      Timestamp<\/p>\n<\/li>\n

    5. \n

      Integrations(where you want to send data i.e Intercom)<\/p>\n<\/li>\n

    6. \n

      Events<\/p>\n<\/li>\n

    7. \n

      Properties<\/p>\n<\/li>\n<\/ol>\n

      Setting Events & Properties for Mixpanel<\/strong><\/h3>\n

      You can send any information about the user as people properties or track their historic activities. Read this article<\/a> on how to use the tool for importing data into Mixpanel. It also has a few simple csv’s to help you get started quickly.<\/p>\n

      Once you upload this file to Segment CSV Importer, this would be then sent to Mixpanel through Segment. <\/p>\n
