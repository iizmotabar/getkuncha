Google analytics custom dashboards contain the most important data that is frequently viewed. Every view in analytics has a by default dashboard, which has some prebuilt widgets. The dashboards created by you are called custom dashboards.<\/span><\/p>\n

You can export, email and share these dashboards. You can either create custom dashboards or import using a template link or from <\/span>analytics solution gallery<\/span><\/a>. <\/span><\/p>\n

You must have the view level access to create and customize google analytics custom dashboards. <\/span><\/p>\n

\u00a0<\/p>\n

How to Create Custom Dashboards?<\/strong><\/h2>\n

Follow the steps below to create your own custom dashboard.<\/p>\n

Step#1:<\/strong> Navigate to your view and select the <\/span>reporting tab.<\/span><\/p>\n

Step#2:<\/strong> Select Dashboards and click New Dashboard.<\/span><\/p>\n

Step#3:<\/strong> From the pop-up window, click on either the blank canvas or starter dashboard. <\/span><\/p>\n

create custom dashboards<\/p><\/div>\n

How to Import Google Analytics Custom Dashboards?<\/strong><\/h2>\n

Follow the below steps to import custom dashboards to your analytics account. <\/span><\/p>\n

Step#1: <\/strong>Sign in to your analytics account.<\/span><\/p>\n

Step#2: <\/strong>click on the custom dashboard template <\/span>link<\/strong>.<\/span><\/p>\n

Step#3: <\/strong>Select the <\/span>view<\/strong> in which you want to import.<\/span><\/p>\n

import dashboards<\/p><\/div>\n

Step#4: <\/strong>click on the <\/span>Create<\/strong> button.<\/span><\/p>\n

Now click on the dashboard column in your reporting tab to view this dashboard. <\/span><\/p>\n

\u00a0<\/p>\n

How to Share Google Analytics Dashboard?<\/strong><\/h2>\n

Open the dashboard which you want to share.<\/span><\/h2>\n

Click on the share button from the top navigation menu and select if you want to share via template link or to analytics solution gallery. <\/span><\/p>\n

share dashboards<\/p><\/div>\n

How to Add a Widget to Google Analytics Dashboard?<\/strong><\/h2>\n

A dashboard can contain at least 12 widgets. Follow the steps below to add widgets to dashboard.<\/span><\/h2>\n

Step#1: <\/strong>Inside your new dashboard, click on the <\/span>new widget<\/strong> button from the top menu. <\/span><\/p>\n

add widget<\/p><\/div>\n

Step#2: <\/strong>In the next pop up window, enter the <\/span>title<\/strong> for your widget. Select between the real time and standard reports. <\/span><\/p>\n

Step#3: <\/strong>Select the <\/span>dimensions and metrics<\/strong> for the widget. These change depending on the type of widget you selected. <\/span><\/p>\n

Step#4: <\/strong>you can apply <\/span>filter<\/strong> to this widget to only include or exclude specific data. <\/span><\/p>\n

Step#5: <\/strong>you can link this widget to a <\/span>report or url<\/strong>, which makes the widget title a link. This can take you to a specified link or report.<\/span><\/p>\n

Step#6: <\/strong>Click on the <\/span>Save<\/strong> button to view this report.<\/span><\/p>\n

\u00a0<\/p>\n

How to Edit and Delete Analytics Dashboard<\/strong><\/h2>\n

To edit an existing widget, hover over the widget and click on the pen icon appearing at the top right. To delete this widget click on the cross icon.<\/span><\/p>\n

edit\/delete widget<\/p><\/div>\n

How to Clone a Google Analytics Widget<\/strong><\/h2>\n

You can clone widgets in google analytics to make an exact copy. This helps to use one widget as a base for another.<\/span><\/h2>\n

Hover over the widget and click the edit icon. In the edit window click on the <\/span>clone widget<\/strong> button. <\/span><\/p>\n

clone widget<\/p><\/div>\n

How to Add Segments to Dashboard?<\/strong><\/h2>\n

To add a segment to your dashboard, click on the <\/span>add segment<\/strong> button inside the specific dashboard. <\/span><\/p>\n

add segment to dashboard<\/p><\/div>\n

How to Delete Dashboards in Google Analytics<\/strong><\/h2>\n

Open the specific dashboard which you want to delete. On the top right, click on the <\/span>delete dashboard<\/strong> button.<\/span><\/h2>\n
