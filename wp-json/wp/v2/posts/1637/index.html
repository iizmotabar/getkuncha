Google analytics user flow reports are helpful in finding where the user went after a specific page. In order to get to user flow reports navigate to <\/span>audience >> user flow. <\/strong><\/p>\n

<\/p>\n

To check the user flow from the specific page click on the specific page and select \u201cExplore traffic through here<\/strong>\u201d. This will only include the traffic to and from that page. <\/span><\/p>\n

<\/p>\n

User flow can only be seen for landing pages so make sure that the page you are looking for is one of them or else you might need to reach down to that page and select Highlight traffic through here<\/strong>.<\/span><\/p>\n
