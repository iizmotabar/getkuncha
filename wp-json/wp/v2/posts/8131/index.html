Intercom allows you to import historic user data from three different sources<\/a>. If you have your user\u2019s historic data in CSV format, it\u2019s more easier with the tool Segment CSV Importer<\/a>.<\/p>\n

The tool is built on top of segment.com<\/a> and you can use it to create a CSV of any data you want to upload and send it one go. If you are already using segment, the setup is even faster.<\/p>\n

To import your user historic data to Intercom, read the instructions below.<\/p>\n

Note: <\/strong>Make sure to enable Intercom integration<\/a> in Segment before uploading and sending data to Segment.<\/p>\n

Step#1: Visit Segment CSV Importer<\/a><\/strong><\/p>\n

<\/p>\n

Here you would need to have two things:<\/p>\n

    \n
  1. \n

    The write key(find inside your segment account)<\/p>\n<\/li>\n

  2. \n

    CSV file<\/p>\n<\/li>\n<\/ol>\n

    Step#2:<\/strong> Enter the write key into the write<\/strong> field<\/p>\n

    Step#3:<\/strong> Click on the choose button to upload your CSV file containing data<\/p>\n

    Once the file is uploaded click on the import button. This will then send the data to Segment from where it will be forwarded to Intercom.<\/p>\n

    Your CSV file should have the following fields:<\/p>\n

      \n
    1. \n

      Action<\/p>\n<\/li>\n

    2. \n

      User ID<\/p>\n<\/li>\n

    3. \n

      Anonymous ID<\/p>\n<\/li>\n

    4. \n

      Timestamp<\/p>\n<\/li>\n

    5. \n

      traits.property_name<\/p>\n<\/li>\n<\/ol>\n

      To understand more about what each field means read the detailed article here<\/a>.<\/p>\n

      Once you upload this file to Segment CSV Importer, this would be then sent to Intercom through Segment.
