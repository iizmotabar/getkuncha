Facebook custom conversions allow us to track and optimize every action on specific pages of our website. It eliminates the need of adding any extra code to Facebook pixel code. The new Facebook pixel requires a single pixel on every converting page rather than adding a unique pixel for every page.<\/p>\n

In a nutshell, custom conversions made the life of marketers a lot flexible with the new Facebook pixel.<\/p>\n

 <\/strong><\/p>\n

How to Add Custom Conversions<\/strong><\/h2>\n

The foremost thing to do before setting up custom conversions is to install facebook pixel<\/a> on your website. If you already added, follow the below steps to create custom conversions: <\/p>\n

Step1:<\/strong> Click here to open custom conversions<\/a>.<\/p>\n

Step2: <\/strong>Click create conversions<\/p>\n

<\/p>\n

Step3:<\/strong> A pop up window will appear. Here we will add our conversion page url and select a category. All our website pages contain Facebook pixel but conversions don\u2019t happen on every page. For instance, if we want to track our contact us page, we will have to add the URL of thank you page that will be displayed when a visitor completes a contact us form.<\/p>\n

A conversion is recorded when a specific action is completed. Let\u2019s add our thank you page and select a category. You might be tracking a different conversion so make sure to change it.<\/p>\n

 Here rule is a condition to select the page on which you want to add the conversion. One important thing to understand here, if you select URL equals make sure to add the full URL of your page (i.e https:\/\/example.com\/thankyou.html). If you select URL contains, make sure to add a part of URL not the full path.<\/p>\n

<\/p>\n

Once you add the URL and category click on the next button. <\/p>\n

Step4: <\/strong>Give custom conversion a name and description.<\/p>\n

<\/p>\n

Here the description part is optional so make sure to add a descriptive name.  <\/p>\n

If you selling any ecommerce product or want to add value for each conversion check the set a conversion value checkbox and enter the value.<\/p>\n

Step5: Now you will see a verification message for your new custom conversion. Click on the done button to continue.<\/p>\n

<\/p>\n

Using the above website conversion objective you can create your ad. Select your marketing objective as conversions, give your campaign a name, enter your account info and choose your conversion.<\/p>\n
