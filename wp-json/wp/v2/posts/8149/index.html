If you have just redesigned your website and would like to move from .co.uk to .com<\/p>\n

For Instance: https:\/\/abc.co.uk<\/span><\/a> to https:\/\/abc.com<\/p>\n

First,  make sure you have all the proper checks for migration and have prepared the redirect map and everything.<\/p>\n

Second, a few questions naturally come to mind;<\/p>\n

    \n
  1. \n

    How do we switch Google Analytics from an old domain to a new one, without losing GA data while still making the whole process seamless?<\/p>\n<\/li>\n

  2. \n

    What are the best practices for submitting to Search Console?<\/p>\n<\/li>\n<\/ol>\n

    Consider the following as an answer.<\/p>\n

    If you wish to continue tracking with the same GA property for sake of continuity, simply change the website URL on the Property Settings page and on the Settings page of each view. Finally, add an annotation in the GA interface notifying what the change was, and the corresponding date.<\/p>\n

    Follow these steps to change the property name and URL within Google Analytics accounts.<\/p>\n

      \n
    1. \n

      Sign in to your Google Analytics account and go to the current Google Analytics Property you\u2019d like to transfer the domain name for.<\/p>\n<\/li>\n<\/ol>\n

      <\/p>\n

      2. Tap on the Admin panel at the bottom of the main menu.<\/p>\n

      <\/p>\n

      3. Find the property name and default URL in the Property Settings section.<\/p>\n

      <\/p>\n

      4. Replace the Property Name<\/strong> with the new property and the Default<\/strong> URL<\/strong> too.<\/p>\n

      <\/p>\n

      5. Go to View <\/strong>and click View Settings<\/p>\n

      <\/p>\n

      6. Replace View name and Website URL if you have multiple properties. similarly you need to change each and every view setting with the new name and Website URL.<\/p>\n

      <\/p>\n
