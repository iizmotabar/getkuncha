If you\u2019re into analytics, which you obviously are, you\u2019ve probably heard of Google Adwords and Conversion tracking before, however, not necessarily in the same sentence. In this article, we will talk about conversion tracking in Adwords, and I\u2019ll walk you through the four ways to set up conversion in Google Adwords.<\/p>\n

First things first, why would anyone set up conversion tracking in Google Adwords? Good question.<\/p>\n

Quick answer: Mostly because it helps you measure the performance of your marketing campaigns. Without tracking conversions, one wouldn\u2019t know which campaigns are doing well and which ones are not doing so great.<\/p>\n

Moving on. Let\u2019s talk some more about Adwords Conversion Tracking.<\/p>\n

Getting Comfy With Adwords Conversion Tracking <\/strong><\/h2>\n

Adwords conversion tracking involves generating an HTML code that is placed on your website depending on the type of conversion you are tracking. Generally there are four types of conversions in Adwords and we will talk \u00a0about setting up each type in this article..<\/p>\n

To access conversions in Adwords, log in to your Adwords account, click on the settings icon and click conversions under measurement.<\/p>\n

Found it?<\/p>\n

Now, Click on the plus icon(+) to create a new conversion.<\/p>\n

<\/p>\n

On the next screen, you should see the option to select the conversion type, much like the screenshot below.<\/p>\n

<\/p>\n

Done? Perfect! Now we can talk about when to set up each conversion and how.<\/p>\n

Follow me!<\/p>\n

1. Website <\/strong><\/h3>\n

To set up a conversion on your website, select \u2018website\u2019 as the conversion type. . Your website conversion can be a form submission, a purchase, a sign-up or even a page view. Basically anything within this circle that suits, your fancy.<\/p>\n

For the purpose of explaining how this is done, let\u2019s head on towards a quick example.<\/p>\n

We\u2019re creating a conversion for a form submission on your website, assuming that there is a unique thank you page after the form submission.<\/p>\n

Follow the steps below:<\/p>\n

After selecting the conversion type as \u2018website\u2019, a conversion action needs to be created.<\/p>\n

\u2019 Set the name of your conversion to be Contact Us\/Sign up\u2019 depending on the type of form you are using<\/p>\n

Select the category as \u2018Lead\u2019<\/p>\n

Add a value for your conversion i.e how much a single conversion is worth for your business.<\/p>\n

Depending on your conversion type assign a value for your conversion<\/p>\n

Select the count as \u2018every\u2019<\/p>\n

Conversion window equals \u201830 days\u2019<\/p>\n

View-through conversion window equals \u20181 day\u2019<\/p>\n

Include in conversions equal \u2018yes\u2019<\/p>\n

Attribution model equals last-click<\/p>\n

Look at the image below for an overview<\/p>\n

<\/p>\n

Now if you\u2019re satisfied with how this has turned out, hit the \u00a0\u2018CREATE AND CONTINUE\u2019 button.<\/p>\n

On we go!<\/p>\n

Next,\u00a0you will see options for how you want to set up the conversion code. For the sake of this article, I am choosing the \u2018install the tag yourself\u2019 option. However, if you\u2019re more comfortable with the idea of adding the code using Google Tag Manager, that can definitely be done too, in that case, feel free to hop on over to this article<\/a>.<\/p>\n

<\/p>\n

Now according to the most recent changes in Adwords, you now need to add the global site tag<\/a> to your website before your add the conversion code.<\/p>\n

Now what is Global Site Tag?<\/p>\n

For a short and sweet answer: Global Site Tag is the latest version of the analytics tracking code after universal analytics.<\/p>\n

If you already have gTag added to your site, you may proceed directly to adding the conversion code. If the case is otherwise, however, you will need to replace the old universal analytics, with the gTag code, and then add the conversion code right after.<\/p>\n

<\/p>\n

Now, the next step will depend on your site\u2019s possession of a specific thank you page.<\/p>\n

If you do have a separate, dedicated thank you page, go ahead and select the code to fire on page load.<\/p>\n

In case you don\u2019t have one, you can still fire the code on a submit button-click.<\/p>\n

Now, hit \u2018next\u2019 to finish the set up.<\/p>\n

The next step is to verify if the conversion code is working and the conversions are being recorded in Adwords. To do this you can use Google Tag Assistant<\/a>.<\/p>\n

2. App<\/strong><\/h3>\n

This one tracks conversions on mobile apps. Now, this could be an app installation or a purchase within your app. If you wish to create an app conversion, go ahead and select \u2018app\u2019 as the conversion type.<\/p>\n

Now would be the time to select how you want to track your app conversions i.e firebase, google play, third-party etc, do go through the conversion action steps according to the conversion type you selected.<\/p>\n

A conversion code isn\u2019t an absolute necessity when it comes to app type conversions. You can pretty much get away with simply connecting your Adwords account with the app store, firebase, or any third party store according to the conversion type you selected.<\/p>\n

3. Phone Calls<\/strong><\/h3>\n

Phone calls are very important to your business, especially when you display a phone number on your website or on the ad. If the purpose of your ad or landing page is to generate phone calls, it is important to also track them.<\/p>\n

Google Adwords provides a simple way to track phone calls. All you need to do is set up a phone call conversion. There are three ways to track phone calls in Adwords. Create a new conversion and select the conversion type as phone calls to view these options.<\/p>\n

<\/p>\n

1. Calls from ads using call extension or call-only ads<\/strong><\/p>\n

If you are placing a phone number on any of your ads, you can select this option to measure conversions. To do this, check the first option in the image above and click continue..<\/p>\n

Now, enter the conversion name<\/p>\n

Select the category as \u2018Lead\u2019<\/p>\n

If you want to track the revenue, assign a value to the conversion otherwise, select \u2018do not use value\u2019<\/p>\n

Select the call length. Not every call leads to a conversion but you can select any time frame less than 60 seconds to be a conversion. For example, if you select the call length as 45 seconds, any call greater than 45 seconds will be considered a conversion.<\/p>\n

Select conversion window as 30 days<\/p>\n

Include in conversions equals \u2018yes\u2019<\/p>\n

Attribution model equals last click<\/p>\n

<\/p>\n

Once you have done this, click \u201ccreate and continue\u2019.<\/p>\n

Now this type of conversion doesn\u2019t need a tracking code added to your website. It will automatically record a conversion when a phone call number is clicked on your ad.<\/p>\n

2. Calls to a phone number on your website<\/strong><\/p>\n

This type of conversion is used when you place a phone number on your website. It records a conversion when the phone number on your website is clicked.<\/p>\n

<\/p>\n

Once you fill up the conversion details, click \u2018create and continue\u2019.<\/p>\n

With the new Adwords updates, there are two codes you need to add to \u00a0your website in order to track this conversion. First is the Global Site Tag and second is the conversion code.<\/p>\n

In the second step, choose from the options for Global Site Tag and update it according to your current implementation.<\/p>\n

<\/p>\n

Once you are done, click next\u2019 to complete the conversion.<\/p>\n

3. Clicks on the number on your mobile website<\/strong><\/p>\n

This records a conversion when someone clicks on a phone number link on your mobile website.<\/p>\n

To create this one, follow the same steps as the previous conversion.<\/p>\n

Make sure to install the Global Site Tag first and then add the conversion code in order to track them correctly.<\/p>\n

4. Import<\/strong><\/h3>\n

The final way to setup Adwords conversion tracking is by importing from data sources like Google Analytics, Firebase, Salesforce, third-party apps, or CRMs.<\/p>\n

If you have already set up goals in Google Analytics or any other tool you prefer to work with, you don\u2019t need to create conversions in Adwords. You can simply import those conversions to Adwords.<\/p>\n

One of the best practices \u00a0is to import Google Analytics goals in Adwords. To do this, you will have to connect Adwords with Google Analytics. Then using the import method you can import any goal you have created in Analytics.<\/p>\n
