Do you have a Shopify store? Are you looking to setup enhanced ecommerce tracking? You have come to the right place. In this article, I will show you how to setup Google Analytics enhanced ecommerce tracking in Shopify and give away free pre-built goals and dashboard for your Shopify store.<\/p>\n

Add Google Analytics to Shopify<\/strong><\/h2>\n

Before you set up ecommerce tracking, it\u2019s important that you add Google Analytics tracking ID. In case if you haven\u2019t yet connected analytics with Shopify, follow the steps below, otherwise, you can skip this step.<\/p>\n

    \n
  • Copy the analytics tracking id from your analytics account (admin >> property >> property settings)<\/li>\n
  • Open your shopify dashboard<\/li>\n
  • From the left menu, click \u2018online store\u2019 and then click \u2018preferences\u2019.<\/li>\n
  • Under preferences, paste the analytics tracking id in the Google Analytics field.<\/li>\n
  • Click on the save button.<\/li>\n<\/ul>\n

    Enable Enhanced Ecommerce Tracking<\/strong><\/h2>\n

    To enable enhanced ecommerce tracking, check the \u2018Use Enhanced Ecommerce\u2019 checkbox under the Google Analytics field in your Shopify dashboard.<\/p>\n

    <\/p>\n

    Save the changes.<\/p>\n

    The best thing about Shopify is it takes care of all the ecommerce code and get things done automatically.<\/p>\n

    <\/a><\/div>\n

    Enable Enhanced Ecommerce Tracking in Google Analytics<\/strong><\/h2>\n

    Follow the steps below to enable enhanced ecommerce tracking in analytics.<\/p>\n

      \n
    1. Open your analytics account and navigate to admin<\/li>\n
    2. Select the view in which you want to view ecommerce data<\/li>\n
    3. Under the view options, click E-commerce Settings<\/li>\n<\/ol>\n

      <\/p>\n

      4. Inside enable ecommerce and enhanced ecommerce settings and hit submit.<\/p>\n

      <\/p>\n

      You have now successfully enable enhanced Google Analytics enhanced ecommerce in Shopify. Without setting up ecommerce goals and reports the process would be incomplete.<\/p>\n

      Setting up the Right currency in Google Analytics<\/strong><\/h2>\n

      By default, the currency in Google Analytics is set to the US Dollar ($). If you don\u2019t sell stuff in the US, you will have to change the currency settings in Analytics. To do this, just click \u2018View Settings\u2019, now scroll down to \u2018currency\u2019. Select the correct currency from the drop-down, and you\u2019re done!<\/p>\n

      <\/p>\n

      Setting up Referral Exclusion list in Google Analytics<\/strong><\/h2>\n

      If you are using PayPal or any other payment gateway, you will need to add the payment gateway domains to the referral exclusion list. Doing this will help Google Analytics attribute the conversion to the right traffic source. For Shopify, you also need to add the checkout.shopify.com to your referral exclusion list.<\/p>\n

      <\/p>\n

      Testing Ecommerce Tracking<\/strong><\/h2>\n

      Now, let\u2019s make sure that the ecommerce tracking works correctly. First, open preview mode in Google Tag Manager. Next, in a separate tab, open your website. Now, make a test purchase using your own credit card. You can also use a 100% discount coupon, if that suits you, just go with the easier option.<\/p>\n

      Now you wait. Give it an hour, and then open your Google Analytics account. Select the test view. Now, navigate to Conversion >> <\/strong>Ecommerce<\/strong> >> Overview<\/strong> and set the date range to include today. If you are able to see the transaction data for the product you purchased, you are good to publish the tag manager changes. If however, you can\u2019t see the ecommerce data, check if you\u2019ve followed all the steps correctly, and wait a little more.<\/p>\n

      Once you verify the ecommerce data, create one more view and name it, \u2018Master View\u2019. Enable ecommerce tracking, select currency, exclude bot traffic and use this view for data analysis.<\/p>\n

      Set up Purchase Goal Funnel<\/strong><\/h2>\n

      Setting up the purchase goal funnel help you see how many users drop off during the purchase journey. It helps you see the number of users dropping off on each particular step. This can help you improve the funnel to make sure more users purchase from your shop.<\/p>\n

      We already have built the goal funnel for Shopify and shared it in one of our previous articles. You can click here<\/a> to open the article and import the goal funnel into your analytics account.<\/p>\n
