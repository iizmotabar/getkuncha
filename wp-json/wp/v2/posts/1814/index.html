YouTube video tracking has now become easier since the release of built-in video trigger in Google Tag Manager. If you are looking to track embedded YouTube videos, this guide is for you.<\/p>\n

Without wasting time explaining the same things, let’s face the new video trigger and variables. Follow the steps below:<\/p>\n

Step1: Enable Built-in Video Variables<\/strong><\/h2>\n

Before doing anything specific, click on variables and enable all the new video built-in variables.<\/p>\n

<\/p>\n

Step2: Create a Custom JavaScript Variable to Track Action<\/strong><\/h2>\n

Within the variables section, after enabling the built-in video variables. The second step is to create a user-defined custom javascript variable.<\/p>\n

This variable will capture the action a user takes on your video(play\/pause).<\/p>\n

Click New button under user-defined variables.<\/p>\n

<\/p>\n

Enter the name of variable i.e JS – Get Video Action<\/p>\n

Click variable configuration and select the variable type as custom javascript<\/p>\n

Paste the below code written by Simo Ahava into the custom JavaScript field<\/p>\n

function() {\n  var status = {{Video Status}};\n  switch (status) {\n    case 'start':\n      return 'Start playing';\n    case 'pause':\n      return 'Pause';\n    case 'buffering':\n      return 'Buffering';\n    case 'progress':\n      return 'Reached ' + {{Video Percent}} + '%';\n    case 'complete':\n      return 'Reached the end';\n  }\n}\n<\/pre>\n
<\/p>\n
Save this variable.<\/p>\n
Step3: Create & Configure Video Trigger<\/strong><\/h2>\n
Open triggers tab and click on the new button to create a new trigger.<\/p>\n
Enter a unique name for the trigger, click trigger configuration, and select YouTube Video as the trigger type.<\/p>\n
Configure the trigger as displayed in the image below. You can uncheck the options you don\u2019t want to track.<\/p>\n
<\/p>\n
The first three things in the above image are understood.<\/p>\n
You can track the video progress in two ways. The first is by percentages, and second by time thresholds. Both are used to track how long a video has been played.<\/p>\n
In the advanced section, by checking the \u2018Add JavaScript API support to all YouTube videos\u2019 will fire the trigger on videos that have JS API support.<\/p>\n
Once you configure this, save the trigger.<\/p>\n
Step4: Create YouTube Video Tag<\/strong><\/h2>\n
Now, navigate to tags section and create a new Google Analytics tag.<\/p>\n
Configure the tag as displayed in the image below.<\/p>\n
<\/p>\n
You will use the video variables you have enabled in the step to send event data to Google Analytics.<\/p>\n
Add the trigger we created in the third step to this tag.<\/p>\n
Save tag and open preview mode. Open your web page containing the video in a new tab.<\/p>\n
If you have done the above steps carefully, you the YouTube event will fire.<\/p>\n
<\/p>\n
To see the data in Google Analytics, open real time >> events.<\/p>\n
<\/p>\n
Now that we have confirmed video tracking is working successfully. You can now publish changes in Google Tag Manager.<\/p>\n
