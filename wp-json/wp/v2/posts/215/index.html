      2. How to Setup Google Analytics Account? <\/strong><\/h2>\n\n\n\n

      Step 1: <\/strong>First you would need a google analytics account. For that visit <\/span>https:\/\/analytics.google.com<\/span><\/a>. If you already have a Gmail account, use that to signing with. If you don\u2019t have sign up for a new Gmail account for yourself.<\/span><\/p>\n\n\n\n

      Step 2: <\/strong>The next screen will show you options for selecting between the website and mobile app. You will have to select the tracking method as a website.<\/span><\/p>\n\n\n\n
