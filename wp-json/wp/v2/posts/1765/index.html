Step1: Install the Injector Chrome Extension<\/a>. This extension runs your code snippet on any website.<\/p>\n

Step2: Open any website you want to test your GTM code. For instance, We want to test the scroll tracking tags just created in my new tag manager container.<\/p>\n

Step3: Click on the injector icon to open this extension.  <\/p>\n

Step4:  On the injector window, click on the plus(+) icon below the snippets text. On the pop up box, enter the domain name(i.e marketlytics.com just the domain name not the full url) and click on create button.<\/p>\n

Step5: copy the GTM container code and paste into the javascript code area. Remove the script tags from the code and don\u2019t remove any empty line before the code. Click on the save button. Your code should appear like<\/p>\n

Step6: From your tag manager account enter the preview mode and open the website in a new tab.
