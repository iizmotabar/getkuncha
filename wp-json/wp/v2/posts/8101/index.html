In this post we will run through two common use case for google tag manager. Implementing google analytics page tracking and adwords conversion tracking using gtm.<\/em><\/p>\n

Using Google Tag Manager to implement Google Analytics<\/h2>\n

Google Tag Manager is a new solution by google that allows you to create a single script tag to all the site pages and then deploy additional tracking code to the site directly from gtm’s web interface.<\/p>\n

Setting up GTM<\/h2>\n

To begin with GTM you will need to visit https:\/\/google.com\/tagmanager<\/a>. Once you login, there will be an option to create a new tag manager account or use an existing one. For simplicity you should create a new account for each of your web properties.<\/p>\n

    \n
  • Give the account and container a name<\/li>\n
  • Select the timezone (ideally same as the GA account)<\/li>\n
  • Add domain name(s)<\/li>\n
  • Click create account<\/li>\n<\/ul>\n

    On the next page you will see a script tag. This is the code for the accounts GTM container and needs to be added on all pages of site after<\/p>\n

    Below the code you will see an option to create a GA tag. Clicking this will take you to a new page where you can setup GA tracking for the site.<\/p>\n

    This is how it would l<\/a>ook<\/p>\n

    Add a new tag<\/p><\/div>\n

      \n
    • Give the Tag a name<\/li>\n
    • Enter the accounts UA code<\/li>\n
    • Specify Track Type to Page View<\/em><\/li>\n
    • Click Rule to Fire Tag<\/em> in the dialog select all pages.<\/li>\n
    • Hit Save<\/li>\n<\/ul>\n

      \u00a0<\/p>\n

      \u00a0<\/p>\n

      \u00a0<\/p>\n

      \u00a0<\/p>\n

      \u00a0<\/p>\n

      Your tag is done. Next thing is to deploy the tag.<\/p>\n

      Click Create Version button on the Overview screen to push this tag to the container.<\/p>\n

      Next you can click Publish<\/strong> to push changes live.<\/p>\n

      And Voila we have started using GTM for google analytics tracking.<\/p>\n

      \n

      Caution:<\/strong> Before you publish any GA code present on the site using the standard tag should be removed. Container script should be present on all pages.<\/p>\n<\/blockquote>\n

      Now the real deal<\/h2>\n

      The actual power of gtm comes from its ability to allow us to add multiple tracking scripts and trigger them on specific pages based on predefined condition.<\/p>\n

      \n

      Rules tell gtm when a tag should be run.<\/p>\n<\/blockquote>\n

      Imagine setting up new goals and virtual pageviews by just setting up new tags or adding adwords remarketing \/ conversion tags to specific pages without a single code change that is now possible.<\/p>\n

      Before you dive in to details you should read an overview to google tag manager<\/a><\/p>\n

      So how would this work?<\/h2>\n

      There are pre built templates available in gtm for the tags of different google services. Let’s run through a common scenario for triggering an adwords conversion tag:<\/p>\n

      From the overview page you can create new tags. Select New Tag<\/em><\/p>\n

      On the next page you will see a blank tag. Here you can choose from a variety of tag types. For Adwords conversion tracking we will select Adwords Conversion Tracking<\/strong> You will now see the template for an adwords conversion tag loaded. You will see 3 fields which need to be filled.<\/p>\n

      \n

      At this point go to the adwords account and generate a conversion tag. This is what it would look like<\/p>\n

        Now comes the time to specify the rule this is how you tell GTM when to trigger the tag. Click Add Rule to Trigger Tag<\/em><\/p>\n

        On the next dialog we want to select Create a new rule<\/em>:<\/p>\n

        • Give the rule a name<\/li>\n
        • Select what conditions need to be met for the rule to trigger<\/li>\n<\/ul>\n

            \n
          • Most common condition is to specify a url to trigger the tag. This will likely be url of thank you page.<\/li>\n
          • We can also go a step further and make sure only user coming from a specific page is counted as converted.<\/li>\n
          • To do this click the plus sign and add another condition<\/li>\n
          • Select {{referrer}}<\/strong> contains<\/em> then specify the page url a person most come from to be counted as converted.<\/li>\n<\/ul>\n<\/blockquote>\n
              \n
            • Once done hit save to add the rule.<\/li>\n
            • Now save the tag.<\/li>\n<\/ul>\n

              You will now be back at overview screen. Select Create Version<\/strong> to add changes to container.<\/p>\n

              Now click Publish to make this new version of container live on the site.<\/p>\n

              And we are done!<\/h2>\n
