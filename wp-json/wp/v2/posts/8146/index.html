Google Analytics contains a lot lot of data. It is up to you, how you understand and use to improve your business. One of the great use of Google Analytics data is to create an audience for your Adwords ads.<\/p>\n

Audiences in Google Analytics are a group of users who perform similar actions. You can display targeted ads to each audience or display ads based on user behavior which is known as dynamic remarketing.<\/p>\n

To create an audience in analytics, first, you will have to connect Adwords account with analytics.<\/p>\n

How to Create Audience in Analytics<\/strong><\/h2>\n
    \n
  1. To create your first audience in Google Analytics, navigate to admin >> property.<\/li>\n
  2. In the property column scroll down to audience definitions >> audiences.<\/li>\n
  3. Analytics by default lets you create your first audience which is All Users.<\/li>\n
  4. To create the first audience, select the audience source( select views)<\/li>\n<\/ol>\n

    <\/p>\n

    Google will automatically enable remarketing feature in your analytics account if it is turned off. Click next button from the enable remarketing option.<\/p>\n

    <\/p>\n

    Google Analytics allows you to use your audience for one source at a time. To select your audience destination, click on the audience destination and select the Adword account in which you want to use this audience.<\/p>\n

    <\/p>\n

    Click on enable button to use this audience. As described earlier this will be your all users audience. You can use this to target all users who visited your website in the last 30 days.<\/p>\n

    Note: you can publish a maximum of 20 audiences to Google Analytics.<\/strong>You can create upto 2000 audiences per property.<\/strong><\/p>\n

    Below are the few of the predefined audiences you can use in analytics.<\/p>\n

    <\/p>\n

    If you require a different audience, you can create an audience yourself or import a segment.<\/p>\n

    Let\u2019s create a new audience and also import one from segment.<\/p>\n

    Create audiences for users who visited your site using mobile device, using chrome browser, age between 25-34 and are located in Hong Kong<\/strong><\/h3>\n

    1. Follow the steps below to create the audience<\/li>\n

      <\/p>\n

      Our first condition here is to select users who have visited from mobile device using chrome browser.<\/p>\n

      From the audience builder, select technology column and select the device category contains mobile and browser contain chrome.<\/p>\n

      <\/p>\n

      Next, we have to select those who are located in HK and the age group is 25-34<\/p>\n

      Select demographics column from the audience builder, select age group and country as Hong Kong.<\/p>\n

      <\/p>\n

      Click on ‘apply’ button.<\/p>\n

      Select the membership duration. You can select this between 1 to 540. This will tell how long a user will be included in this audience.<\/p>\n

      <\/p>\n

      Enter audience name and click next step button.<\/p>\n

      Select the destination and publish audience.<\/p>\n

      Create audience by importing audience<\/strong><\/h3>\n

      Now, we will create an audience by importing a segment.<\/p>\n

      Follow the steps below to create the audience:<\/p>\n

        \n
      1. Click new audience button within the audience section<\/li>\n
      2. Select audience source(view) the way we did above<\/li>\n
      3. From audience definition, click on Import Segment button<\/li>\n<\/ol>\n

        <\/p>\n

        Next, select the segment which you want to import. In my case, I am selecting performed site search segment(users who performed a site search).<\/p>\n

        <\/p>\n

        Next select membership duration, enter audience name and click next to select destination and publish audience.<\/p>\n
