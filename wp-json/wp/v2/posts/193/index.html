      What is Bing Ads UET Tag?<\/h2>\n

      It tracks what your customers are doing after they click on your ad. Universal Event Tracking (UET) is a powerful tool that records what customers do on your website. By creating one UET tag and placing it across your website, Bing Ads will collect data that allows you to track conversion goals and target audiences with remarketing lists -advertise.bingads.microsoft.com.<\/em><\/p>\n

      Bing Universal Event Tracking<\/p>\n

      In short, UET tags provide you essential data to help you improve ad performance on your website.<\/span><\/p>\n

      Creating Bing Ads UET Tag<\/h2>\n

      It is easy to create a UET tag in Bing Ads. All you need is a name<\/span><\/p>\n

      create UET tag<\/p>\n

      1. Within Bing ads portal Click the Campaigns<\/strong> page, and then on the left pane, click Conversion Tracking<\/strong>.<\/p>\n

      2. Under Conversion Tracking, click UET tags<\/strong>.<\/p>\n

      create UET Tags<\/p>\n

      Note:<\/strong> If this is the first time you are creating a tag, you can also click on <\/span>Get Started<\/strong> in the <\/span>Conversion Tracking page.<\/strong>Click<\/p>\n

      3. Create UET tag<\/strong> and then enter a name.<\/p>\n

      Optional: Enter a description that helps you identify the UET tag in the future. We recommend that you enter the website page name or URL.<\/span><\/p>\n

      4. <\/span>Click Save<\/strong>.<\/p>\n

      5. In the View UET tag tracking code box, click either Copy, Download or Email, and then click Done.<\/p>\n

      UET tag tracking code<\/p>\n

      Setting up Bing Ads UET Tag in Google Tag Manager<\/h2>\n

      Once you have created Bing UET Tag, setting it up using GTM is easy.<\/span><\/p>\n

      1. Create a NEW tag in GTM by clicking on the <\/span>NEW<\/strong> button within the <\/span>Tags<\/strong> section of your container.<\/span><\/span><\/p>\n

      Create a new GTM tag<\/p>\n

      2. On the popup window, enter the title of the tag as \u201cBing UET Tracking<\/strong>\u201d<\/p>\n

      3. <\/span><\/span>Click on the tag configuration and select the tag type as \u201cBing Ads Universal Event Tracking<\/strong>\u201d.<\/p>\n

      4. In the tag configuration menu, enter the UET tag id we created earlier, keep the UETQ variable as it is and select the event type as page load.<\/p>\n

      <\/p>\n

      5. Click on the advanced options and set the tag firing option as \u201conce per page\u201d.<\/strong><\/p>\n

      <\/p>\n

      6. Now select All Pages<\/strong> within Fire On<\/strong> heading as the trigger condition for the tag. This will add the code to all pages of site. Click on the Save<\/strong> button. Once GTM is published<\/a> the code will be immediately added to your site and tracking will begin.<\/p>\n

      Bing Events<\/h2>\n

      Bing Ads allow you to report custom events (such as downloading whitepapers, subscribing to newsletters etc) from UET and then choose a subset of those custom events to set up as conversion and remarketing list. They require that you report values for one or more of the following custom parameters when custom events happen and create custom event type goals and when audience specifies which values for these custom parameters would qualify the custom event as a conversion:<\/span><\/p>\n

        \n
      • Event category<\/strong> – The category of event you want to track. For example: ‘video\u2019.<\/span><\/li>\n
      • Event action<\/strong> – The type of user interaction you want to track. For example, ‘play’ or ‘pause’ etc.<\/span><\/li>\n
      • Event label<\/strong> – The name of the element that caused the action. For example, ‘trailer’ or ‘behindthescenes’, etc.<\/span><\/li>\n
      • Event value<\/strong> – A numerical value associated with that event. Could be length of the video played, etc.<\/span><\/li>\n<\/ul>\n

        Similar to UET Tag, you can also set up bing events in Google Tag Manager and then later use event values to setup custom audience and different conversions to enhance your ads setup. Bing events are similar to Google Analytics events as they collect four different data variables similar to GA i.e category, action, label and value.         el   = event label (Home Page)<\/span><\/p>\n

        What is Bing Ads Remarketing?<\/h3>\n
        Bing ads remarketing is a second opportunity to bring back the customers who leave without purchasing. It\u2019s kind of reconnecting and reminding people to come back and complete the purchase. Remarketing in paid search delivers your ads to previous visitors when they search via Bing and Yahoo.<\/span><\/p>\n
