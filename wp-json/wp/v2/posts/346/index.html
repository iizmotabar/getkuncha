We have witnessed many of updates since Looker Studio has been launched globally. One of the important updates is the addition of third-party data connectors. With third-party connectors, we can now import data from many of the external marketing channels like Facebook, Bing, DoubleClick..etc.<\/p>\n

In this article, we will discuss why you should say goodbye to Google Sheets and hello to third-party data connectors.<\/p>\n

The conventional data import method<\/strong><\/h2>\n

Before data connectors, Google Sheets were used as a medium to import data from other sources. It is still a workaround practiced by many of the online marketers but is a complex process.<\/p>\n

With Google Sheets, you have to gather all your data in a single sheet and have to be careful about each detail, to be able to import data to Looker Studio successfully.<\/p>\n

Another painful workaround with sheets is, you have to make sure the date and time formats are compatible with Looker Studio. Following are the four important things that make Google Sheets less useful.<\/p>\n

    \n
  1. Create a new sheet and gather data<\/li>\n
  2. Define a process to update data<\/li>\n
  3. Looker Studio lacks real-time reports<\/li>\n
  4. Make sure to make date and time compatible<\/li>\n<\/ol>\n

    <\/h2>\n

    Data Connectors: The Perfect Sheet Replacement<\/strong><\/h2>\n

    Looker Studio introduced third-party connectors to eliminate the need of sheets to import data from other marketing channels. With data connectors, it became a very simple one-step process to visualize data from any source.<\/p>\n

    Looker Studio has now over 10 partners, where Funnel<\/a>\u00a0made a super connector that collects data from 250+ sources and send to 100+ destinations.<\/p>\n

    How to use data connectors<\/strong><\/h3>\n

    Connecting a third party data source is similar to how we connect native data sources<\/a>. However, connecting a third party data connector can be a pretty intimidating process at first for most of the people like me.<\/p>\n

    Let me show you how to connect Facebook Ads connector by Supermetrics to build a report like I did. Follow the step by step process:<\/p>\n

    Step1:<\/strong> Create a new report or open an existing report to view you want to add the data source<\/p>\n

    Step2:<\/strong> click on CREATE NEW DATA SOURCE<\/strong> button on the right menu when created a new report or click Resources >> Manage added<\/strong> and then click Add a Data Source<\/strong> link for any existing report.<\/p>\n

    <\/p>\n

    Step3:<\/strong> From the left menu scroll down to see the community connectors and click EXPLORE CONNECTORS<\/strong> button.<\/p>\n

    <\/p>\n

    Step4:<\/strong> Here you will see a list of available third-party connectors. Look for Facebook Ads by Supermetrics<\/strong> connector and click ADD CONNECTOR<\/strong> link.<\/p>\n

    <\/p>\n

    Step5:<\/strong> Authorize Looker Studio to connect to use the community connector. Click Authorize<\/strong> button<\/p>\n

    <\/p>\n

    Step6: <\/strong>Now authorize Facebook Ads connector<\/p>\n

    <\/p>\n

    Follow the steps ahead to sign into facebook account
    \nStep7:<\/strong> select ad account, conversion window<\/strong> and click on CONNECT<\/strong> button on top right.<\/p>\n

    <\/p>\n

    Step8: <\/strong>click on Add to Report<\/strong> button to import metrics and dimensions<\/p>\n

    <\/p>\n

    Once you follow the steps successfully you can now create your report. Explore more about Supermetrics connectors<\/a> and how you can use them together to build a single report for all your marketing data.<\/p>\n
