Activity Repeat Time in google analytics shows the times at which a user repeated the same activity within a single session. <\/span><\/p>\n

For example, if a user has viewed a single page 3 times during the same session, activity repeat time will show the timings at which the activity is repeated.  <\/span><\/p>\n

To look at the activity repeat time, navigate to <\/span>Audience >> User Explorer<\/strong>. <\/span><\/p>\n

<\/p>\n

You will see the list of user IDs who have visited your site. <\/span><\/p>\n

<\/p>\n

Click any of the client IDs and you will be able to see the the total number of sessions listed. <\/span><\/p>\n

<\/p>\n

\u00a0<\/p>\n

Click any of the sessions to see which pages a user viewed during the session, the number of goals completed, the number of events and if there were any e-commerce transactions. <\/span><\/p>\n

<\/p>\n

\u2018Last view\u2019 is the time when a user last viewed the page.\u2019Activity repeat\u2019 is the number of times the page is viewed and the \u2018activity repeat time\u2019 shows the previous times at which the same page was viewed.<\/span><\/p>\n
