Enabling site search in google analytics will give you strong keywords for on site optimization. In google analytics you can view the search terms that your audience use to find your content.<\/p>\n

 <\/p>\n

How to track site search in Google Analytics?<\/strong><\/h2>\n

You might have noticed and if not do it now. Open our analytics-faq <\/a>page in a new tab and in the search bar type google analytics and click on the first page. Now check the URL of the page. You might see this post at first.<\/span><\/p>\n

https:\/\/marketlytics.com\/analytics-faq\/delete-google-analytics-shared-account<\/span>?rq=google%20analytics<\/span><\/em><\/strong><\/p><\/blockquote>\n

You can see the search query is attached to the end of actual url separated by question mark(?). <\/span><\/p>\n

Google analytics tracks the url of the page checks for any parameters attached to the actual string and it by default differentiate between if it is a search term or not. This way you can get the search terms being entered on your site search bar. <\/span><\/p>\n

But first, you would have to enable site search in google analytics. <\/span><\/p>\n

\u00a0<\/p>\n

How to enable site search in google analytics?<\/strong><\/h2>\n

In your analytics account navigate to the specific view<\/strong> in which you want to enable site search<\/p>\n

Next, click view settings<\/strong><\/p>\n

open view settings<\/p><\/div>\n

Under site search<\/strong> settings, switch on the site search tracking<\/strong><\/p>\n

In the query parameter box you can write searchQuery<\/strong> to identify the query url in your reports<\/p>\n

Now your setup should look something like this<\/p>\n

site search google analytics<\/p><\/div>\n

Finally, save the changes by clicking the save button.<\/p>\n

Once you enabled site search it may take upto 24 hours to generate search reports in your GA account.<\/p>\n

\u00a0<\/p>\n

How to see site search terms in google analytics?<\/strong><\/h2>\n

To see the search reports in your analytics account you should navigate to Behavior >> Site Search >> Overview<\/strong><\/p>\n

view site search reports<\/p><\/div>\n
