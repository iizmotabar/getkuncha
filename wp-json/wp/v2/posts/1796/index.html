If you are looking to install Google Tag Manager on Shopify, you are on the right place. In this guide, I will show you the step by step instructions to integrate Shopify with Google Tag Manager.<\/p>\n

Why Tag Manager for Shopify?<\/strong><\/h2>\n

Google Tag Manager is the most popular and widely used tag management solution for online ecommerce stores and websites. The benefits of using GTM are countless but I am only going to highlight a few essential benefits for you to get you started.<\/p>\n

    \n
  1. Installing GTM on your ecommerce site improves your site speed. Because this is a tag management solution, flexible enough to store all your analytics and marketing tags. You have the control to fire tags when needed. Another part it plays to speed up your site is by calling all external files in one go. For instance, if you have multiple marketing tags in store, this will call them once without waiting for other tags to fire.<\/li>\n
  2. You can set up Google Analytics event tracking without modifying code on your site. Hence, you don\u2019t have to depend on a developer to implement code changes each time you set up new event tracking.<\/li>\n
  3. GTM contains built-in tag templates for Google Analytics, Adwords, Facebook Pixel and more. It also works with few analytics and tracking tools.<\/li>\n
  4. If you need to add any other script on your site you can use custom HTML tag without having to touch with your backend code.<\/li>\n<\/ol>\n

    Getting Started<\/strong><\/h2>\n

    First, you will have to create a Google Analytics account for your Shopify store. if you haven\u2019t already done so. You can follow add Google Analytics to Shopify<\/a> instructions for assistance. Make sure to follow just  the \u201cHow to setup Google Analytics Account\u201d topic and stop when you get to the tracking code. As we will be replacing  the GA tracking code with  tag manager.<\/p>\n

    Setting up Google Tag Manager <\/strong><\/h2>\n

    Now you need to configure Google Tag Manager. Go to tagmanager.google.com<\/a> and sign in using the same gmail id you previously used for Google Analytics.<\/p>\n

    After logging in, you will be directed to add a new account. Just fill in the name of your account and click the continue button.<\/p>\n

    <\/p>\n

    The next step is to add a container to your account. A container could be a website or a mobile app but in this case, we are adding a website.<\/p>\n

    If you are not sure about the container name, simply enter your website\u2019s domain. Select \u2018web\u2019 to use this container for your ecommerce site.<\/p>\n

    <\/p>\n

    Once you are done, click on the create button.<\/p>\n

    Then a pop-up window will appear displaying the agreement details. To proceed you have to accept the agreement and a new pop-up will then appear displaying your container code.<\/p>\n

    <\/p>\n

    Click on the ok button to close this pop-up.<\/p>\n

    Add Google Analytics Pageview Tag<\/strong><\/h2>\n

    If you want to execute ecommerce tracking then you should not add this GA pageview tag. Instead, you should add the GA tracking id in your Shopify store and enable ecommerce tracking.<\/p>\n

    If you want to track ecommerce tracking with google tag manager then you will have to add the GA pageview tag inside tag manager. Here is a brief guide on connecting Google Tag Manager with Analytics<\/a>.<\/p>\n

    \n

    Adding Google Tag Manager to Shopify<\/strong><\/h2>\n

    You can add Google Tag Manager container code in two places. First in theme.liquid file and second on the checkout page.<\/p>\n

    Paste the GTM container code on the Theme.liquid file<\/strong><\/p>\n

    Open your Shopify dashboard in a new tab.<\/p>\n

    From the left menu bar, navigate to Online Store >> Themes<\/strong><\/p>\n

    Now, click on the three horizontal dots button and click on Edit HTML\/CSS<\/strong>.<\/p>\n

    <\/p>\n

    Next, click on the theme.liquid file from the and paste the Google Tag Manager container code you previously copied just before the <\/head> <\/strong>tag as shown in the image below.<\/p>\n

    <\/p>\n

    Although it is not necessary to insert the second container code but you can paste it just below the body tag for your satisfaction.<\/p>\n

    Paste the GTM container code on the checkout page (for shopify plus)<\/strong><\/p>\n

    From your Shopify store dashboard, navigate to settings and then click on the checkout tab.<\/p>\n

    Scroll down to see the Additional scripts field.<\/p>\n

    Paste the same GTM container code inside this Additional scripts field.<\/p>\n

    <\/p>\n

    Save your settings. <\/strong><\/p>\n

    Verifying Setup <\/strong><\/h2>\n

    Uptil now, you should have added Google Tag Manager on Shopify but you are not sure yet if followed the steps correctly and if you are receiving the accurate reports in Google Analytics.<\/p>\n

    First, we have to verify if we have added the GTM code correctly to our Shopify store. To do this follow the steps below.<\/p>\n

    Install Google Tag Assistant<\/a> in your browser(Google Chrome would be better)<\/p>\n

    On a new tab open your Shopify store. Click the tag assistant icon and then the enable button.<\/p>\n

    <\/p>\n

    Now reload the same page and click on the tag assistant icon. If you have followed the above mentioned steps carefully, you will see the Google Analytics and Google Tag Manager tags firing successfully on your Shopify store.<\/p>\n

    <\/p>\n

    If you are seeing the tags like above then you don\u2019t have to worry about anything. But, if tag assistant is showing any errors you can scroll back to the top of this article and verify steps.<\/p>\n

    Sometimes, Google Tag Assistant shows errors even if your setup is correct. In this case, you can open your Google Analytics account and check real time visitors to verify.
