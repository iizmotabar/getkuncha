Google Tag Manager(GTM) is a powerful tag management tool you can use to manage your analytics and marketing tags. To be able to benefit from this tool, it is necessary that you know the right way to install it on your website.<\/p>\n

When it comes to WordPress, most site owners become very concerned about where and how to add the GTM tracking code. If you are one of those confused WordPress users, you are about to be taught the right way to install Google Tag Manager on your WordPress site.<\/p>\n

By the end of this article, you will be able to set up GTM on your site correctly and have it ready to be put to use.<\/p>\n

1. Create a GTM Account<\/strong><\/h2>\n

If you are new to GTM, you will have to create a new account. Follow the steps detailed below to create a new GTM account. If you already have one, you may skip this..<\/p>\n

    \n
  1. Open tagmanager.google.com<\/a> in a new tab<\/li>\n
  2. Add the required details to create your account<\/li>\n
  3. Enter your website name in the account name field<\/li>\n
  4. Select your country from the drop-down list<\/li>\n
  5. Click continue button<\/li>\n
  6. Enter your website URL in the container name field<\/li>\n
  7. Click web under where to use container options<\/li>\n
  8. Click create to create your account<\/li>\n<\/ol>\n

    <\/p>\n

    A pop-up window will appear with GTM terms & services. Click agree to continue<\/p>\n

    After this, a second pop up will appear displaying the GTM container codes.<\/p>\n

    <\/p>\n

    Keep this window as it is and open your WordPress dashboard in a new tab.<\/p>\n

    2. Adding Google Tag Manager on WordPress<\/strong><\/h2>\n

    There are two ways to add the GTM tracking code on your WordPress site. You can either add the GTM container code directly to the website or install a GTM plugin.<\/p>\n

    The recommended practice is adding the GTM code directly to the WordPress code. However, in some cases, this doesn\u2019t work.<\/p>\n

    There are few WordPress themes that do not allow the addition of external code and if you have a WooCommerce site, you will have to add the GTM plugin for setting up enhanced ecommerce tracking.<\/p>\n

    Both methods are explained in the following.<\/p>\n

    a) Adding GTM without a Plugin<\/strong><\/h3>\n

    Your first choice should be to add the GTM code without using a plugin. To do this,<\/p>\n

    Open your WordPress dashboard<\/p>\n

    Navigate to appearance<\/strong> >> editor<\/strong> from the left menu<\/p>\n

    <\/p>\n

    Look for header.php<\/strong> file from the list of files on the right<\/p>\n

    If you cannot find the header.php file, look for index<\/strong>.<\/strong>php<\/strong> where you can see the head and body part of your website.<\/p>\n

    Now, navigate to your GTM account and copy the first code and paste it right after the opening the head tag in the header.php file.<\/p>\n

    Copy the second code and paste it right after the opening body tag.<\/p>\n

    <\/p>\n

    Click on the update button to save changes.<\/p>\n

    b) Adding GTM using a Plugin<\/strong><\/h3>\n

    There are a lot of plugins available to set up GTM code. But I recommend you to add any of the below two plugins because they work perfectly with almost all WordPress themes.<\/p>\n

    1. DuracellTomi\u2019s Google Tag Manager plugin<\/a><\/strong><\/p>\n

    Add this plugin, if you have a WooCommerce store and want to set up Google Analytics enhanced ecommerce tracking with GTM.<\/p>\n

    If your website falls into this category, we have already covered in detail setting up enhanced ecommerce with GTM<\/a>.<\/p>\n

    2. DCO Insert Analytics Code plugin<\/a><\/strong><\/p>\n

    Install & activate this plugin.<\/p>\n

    Navigate to Settings >> DCO Insert Analytics Code<\/strong> from the left menu<\/p>\n

    Add the first GTM code in the head section field and the second code in the body section field just after the opening body tag.<\/p>\n

    <\/p>\n

    Save the changes.<\/p>\n

    Use the second plugin on any WordPress site you have. The good thing about this plugin is that it adds the GTM code on the right place without causing issues.<\/p>\n

    3. Verifying GTM Installation<\/strong><\/h2>\n

    There are multiple ways to verify GTM installation on your website. The recommended method is to verify with Google Tag Assistant<\/a>. This is a browser extension used for testing and debugging Google tracking codes.<\/p>\n

    After installing this extension, open your website in a new tab. Click the tag assistant icon and enable it. Now, reload your page.<\/p>\n

    Tag Assistant will display all the tracking codes added on your site. If it shows your GTM tracking id, it means the installation is done properly.<\/p>\n

    <\/p>\n

    You will see the GTM tag displayed in yellow color and showing up an error. This isn\u2019t actually an error, after adding anything in your GTM container, this will disappear automatically.<\/p>\n

    4. Connecting GA with GTM<\/strong><\/h2>\n

    Now, if you have added the Google Tag Manager code on your website, there is no need to add the Google Analytics tracking code. You will now use, Google Tag Manager to set up analytics.<\/p>\n

    If you are already using Google Analytics on your website and moving to GTM, make sure to follow through the GTM migration steps<\/a> to protect your data.<\/p>\n

    If you are adding a new Google Analytics account to your website, follow the steps below:<\/p>\n

    Follow the steps here<\/a> to create a new Google Analytics account, in case you don\u2019t know how to do it.<\/p>\n

    After creating your account, navigate to admin, & click property settings<\/p>\n

    Copy the tracking ID displayed under the property settings<\/p>\n

    <\/p>\n

    Now, jump to your tag manager account<\/p>\n

    Create a new tag<\/p>\n

    <\/p>\n

    Enter the tag name as \u2018GA Pageview – All Pages\u2019<\/p>\n

    Click on the tag configuration, and select universal analytics as the tag type<\/p>\n

    Click on the Google Analytics Settings drop-down, and click \u2018new variable\u2019 option.<\/p>\n

    <\/p>\n

    On the pop-up window, enter the variable name as \u2018GA Tracking ID\u2019, and paste the tracking ID you copied earlier in the Tracking ID field.<\/p>\n

    <\/p>\n

    Save this variable.<\/p>\n

    Note:<\/strong> Once you have created a variable for GA tracking ID, you can use this for all GA tags you create in the future. There is no need to copy and paste the GA ID again next time.<\/p>\n

    Click on the \u2018triggering\u2019 option under the tag configuration and select the \u2018All Pages\u2019 default variable.<\/p>\n

    Your final, Google Analytics tag should look similar to the one below<\/p>\n

    <\/p>\n

    Save this tag.<\/p>\n

    The purpose of Google Analytics is to track the number of people who visit your website, so, it is necessary to exclude your own website visits, or internal traffic, in other words, in order for the data to remain accurate. We have already talked about excluding internal wordpress traffic in detail in a previous article, however, if you missed it, you can check it out here.<\/a><\/p>\n
