Google analytics will track all those pages containing the tracking code. You can have multiple similar pages with a slight change in url or a page with multiple aliases. <\/span><\/p>\n

For example you can have a page url as:<\/span><\/p>\n

\/thank-you.html<\/span><\/p>\n

\/thankyou.html<\/span><\/p>\n

\/folder\/anotherfolder\/example-today.html<\/span><\/p>\n

\/folder\/anotherfolder\/exampletoday.html<\/span><\/p>\n

If you look at the above example you would see that both urls are quite similar but not exactly the same. <\/span><\/p>\n

The best way to view reports for a single page with multiple aliases is:<\/span><\/p>\n

Set up content grouping for similar types of pages and use group names to filter out different pages on site. <\/span><\/p>\n

Here is how to set up content grouping in analytics.<\/span><\/p>\n
