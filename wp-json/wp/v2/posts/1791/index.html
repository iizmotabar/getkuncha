Setting up Google Analytics on a Wordpress site should not be an intimidating process. It\u2019s like adding a few lines of code. I have seen many Wordpress owners getting confused while installing Google Analytics and adding multiple types of plugins and ending up with a messy dashboard and zero results.<\/p>\n

In this guide, I will show you how to install Google Analytics on Wordpress site without any plugin within a minute. I am not going into the details of creating a Google Analytics account because it\u2019s simple and you can do by typing in your browser analytics.google.com.<\/p>\n

How to Add Google Analytics Code on Wordpress<\/strong><\/h2>\n

I am assuming that you have created a Google Analytics account. Follow the steps below to add Google Analytics code on Wordpress.<\/p>\n

Step1: If you don\u2019t know where to find the analytics tracking code, login to your GA account, select the specific property and navigate to admin.<\/p>\n

<\/p>\n

Step2:<\/strong> Under property, click on tracking info >> tracking code<\/strong>.<\/p>\n

<\/p>\n

Step3:<\/strong> copy the tracking code between the <script><\/script> <\/strong>tags.<\/p>\n

<\/p>\n

Step4:<\/strong> Now, open your Wordpress dashboard and navigate to Appearance >> Editor<\/strong><\/p>\n

<\/p>\n

Step5:<\/strong> look for the header.php<\/strong> file and open it.<\/p>\n

<\/p>\n

Step6:<\/strong> Now paste the Google Analytics tracking code just before the <\/head><\/strong> tag and hit the update button.<\/p>\n

<\/p>\n

Step7:<\/strong> Open your website in a new tab and go back to your GA account. Navigate to Real-Time >> Overview<\/strong>. If you have followed the above steps carefully, you will see the number of visitors on your website currently.<\/p>\n

<\/p>\n

In case you don\u2019t see any real time data, move to the top of this article and re-check if you made any mistake. Otherwise, you can install Google Tag Assistant<\/a> extension and look for the errors<\/a> it shows.<\/p>\n
