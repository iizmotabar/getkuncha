If you have setup Facebook pixel on your website, you can easily create a custom audience based on the specific pixel.<\/p>\n

A custom audience is a group of website visitors who perform the same action or view the same page. You can use a custom audience for retargeting and product dynamic remarketing.<\/strong><\/p>\n

For example, if you have an ecommerce website and want to retarget visitors who have clicked on the add to cart button but didn\u2019t purchased, you can create a custom audience.<\/p>\n

To do this, you will have to make sure that you have setup the add to cart pixel to fire when the add to cart button is clicked. Basically, you will have to setup standard and custom event pixels<\/a>.<\/p>\n

After you setup pixel, the data appears in ads manager.<\/p>\n

<\/p>\n

Now, to create a custom audience for visitors who have clicked the add to cart button but didn\u2019t purchased, click on create custom audience button.<\/p>\n

<\/p>\n

From the filter, select event as add to cart to include and select purchase to exclude.<\/p>\n

<\/p>\n

Enter the name of the audience and click on create audience button.<\/p>\n

Once the audience is setup, you can create a new ad and only target the audience you created above. This will help you improve generate more sales.<\/p>\n

Besides retargeting you can also setup dynamic remarketing<\/a> based on the visitor behavior.<\/p>\n
