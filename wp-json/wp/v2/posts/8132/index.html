There are two methods for importing historical data into amplitude. The first is a traditional method<\/a> for all amplitude users and the second is by using the Segment CSV Importer<\/a> tool. This tool is built on top of Segment, so if you are already using Segment, the setup is even faster.<\/p>\n

\u00a0<\/p>\n

Importing Historical Data into Amplitude<\/strong><\/h2>\n

To import the historical data into Amplitude via segment csv importer, you need to have two things.<\/p>\n

    \n
  1. \n

    Your historical data in a CSV format<\/p>\n<\/li>\n

  2. \n

    Write key(find in your segment account)<\/p>\n<\/li>\n<\/ol>\n

    \u00a0<\/p>\n

    Note: <\/strong>Make sure to enable Amplitude integration<\/a> in Segment before uploading and sending data to Amplitude.<\/p>\n

    To upload your historical data follow the steps below.<\/p>\n

    Step 1: Visit Segment CSV Importer<\/a><\/p>\n

    <\/p>\n

    Step 2: Enter the write key into the write field<\/p>\n

    Step 3: Click on the choose button to upload your CSV file containing data.<\/p>\n

    \u00a0<\/p>\n

    Once the file is uploaded click on the import button. This will then send the data to Segment from where it will be forwarded to Amplitude.<\/p>\n

    Your CSV file could have the following fields:<\/p>\n

      \n
    1. \n

      Action<\/p>\n<\/li>\n

    2. \n

      User ID<\/p>\n<\/li>\n

    3. \n

      Anonymous ID (either user id or anon id is required)<\/p>\n<\/li>\n

    4. \n

      Timestamp<\/p>\n<\/li>\n

    5. \n

      Events<\/p>\n<\/li>\n

    6. \n

      Properties<\/p>\n<\/li>\n<\/ol>\n

      \u00a0<\/p>\n

      Setting Events & Properties for Amplitude<\/p>\n

      You can send any information about the user as people properties or track their historic activities. Read this article<\/a> on how to use the tool for importing data into amplitude. It also has a few sample CSV’s to help you get started quickly.<\/p>\n
