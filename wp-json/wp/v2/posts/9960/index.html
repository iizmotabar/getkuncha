About MarketLytics<\/h3>\n\n\n\n

MarketLytics is a user data consultancy and advanced analytics provider, that helps clients understand their user behavior. We collaborate with a network of clients and partners across 26 countries including F500s, fast-growing startups, and global e-commerce brands.<\/p>\n\n\n\n

Job brief<\/h3>\n\n\n\n

We are looking to add Data Engineers with at least 1 \u2013 5 years of experience to the team. The role will require working with other Data Engineers to streamline the development, testing, and deployment of data pipelines and products.<\/p>\n\n\n\n

  • Qualifications and Experience<\/strong>
    • A bachelor\u2019s degree in computer science, engineering, applied math, or a related subject<\/li>
    • Experience with relational SQL and NoSQL databases<\/li>
    • Experience with data pipeline and workflow management tools<\/li>
    • Experience with programming languages\/technologies used in data engineering and data science, at least one of Python, R, or Node.JS<\/li>
    • Experience or basic knowledge of working on cloud technologies like GCP or Azure<\/li>
    • Knowledge or prior experience with extracting and creating pipelines for User Behaviour data\/ Marketing Analytics platforms\/ CRMs\/ ERPs would be a plus<\/li>
    • Prior Dashboarding experience with Tableau\/ PowerBI\/ Redash<\/li>
    • Strong communication skills<\/li>
    • Intellectually curious<\/li>
    • Ability to prioritize and manage work independently<\/li><\/ul><\/li>
    • Key Responsibilities<\/strong>
      • Create and maintain optimal data pipeline architecture<\/li>
      • Construct documentation for ETL operations, object usage, and data models.<\/li>
      • Assemble large, complex data sets that meet functional\/ non-functional business requirements.<\/li>
      • Build the infrastructure required for optimal extraction, transformation, and loading of data from a wide variety of data sources using SQL and Cloud Native \u2018big data\u2019 technologies.<\/li>
      • Work with data and analytics experts to strive for greater functionality in our data systems.<\/li>
      • Designing and building production-grade data pipelines from ingestion to consumption within a hybrid big data architecture, using Python<\/li>
      • Performing detailed assessments of current state data platforms and creating an appropriate transition path to Azure\/ GCP
        <\/li><\/ul><\/li><\/ul>\n\n\n\n
