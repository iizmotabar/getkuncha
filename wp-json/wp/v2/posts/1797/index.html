What are Google Tag Manager Variables?<\/strong><\/h2>\n

Variables in Google Tag Manager are name: value<\/strong> pairs for which the value is populated during runtime. It is a storage location where you can keep the URL of the currently loading page or id of any element.<\/p>\n

There are two ways to use variables in tag manager. You can use them in triggers as a filter to specify when the trigger should execute i.e execute the trigger when the page path variable matches \/thankyou.html. In tags, variables are used to store dynamic values i.e transaction values or conversion tracking tag.<\/p>\n

Types of Variables<\/strong><\/h2>\n

There are mainly two types of Google Tag Manager variables.<\/p>\n

    \n
  1. Built-in Variables<\/li>\n
  2. User Defined variables<\/li>\n<\/ol>\n

    Built-in variables are predefined variables and can not be customized. To view, the list of built-in variables navigate to variables tab.<\/p>\n

    <\/p>\n

    To view and enable the built-in variables click on the configure button.<\/p>\n

    <\/p>\n

    Only a few of the specific variables are enabled by default and you will have to enable the other variables to use them. Following is the list of built-in page variables:<\/p>\n

    Page URL: <\/strong>\u00a0returns the string containing the full URL of the current page (https:\/\/marketlytics.com\/contact\/lets-talk)<\/p>\n

    Page Hostname:<\/strong> returns the string containing the hostname of the currently loaded page (https:\/\/marketlytics.com)<\/p>\n

    Page Path:<\/strong> returns the string containing the relative path of the current page (\/contact\/lets-talk)<\/p>\n

    Referrer:<\/strong> returns the string containing the URL of the previously viewed page by the user (https:\/\/marketlytics.com\/blog)<\/p>\n

    User-Defined Variables in Tag Manager<\/strong><\/h2>\n

    User-defined variables are those variables which are defined by me and you. These variables are other than those listed in built-in variables. To create a user defined variable you can click on the new button under user defined variables.<\/p>\n

    For example, you can define a constant variable to save the GA tracking code, which can be used anywhere when needed the GA code.<\/p>\n

    Click on the New button<\/p>\n

    Enter the name as \u2018gaProperty\u2019<\/p>\n

    Click on the variable configuration and select constant as the variable type<\/p>\n

    Enter the GA tracking code in the value field and hit save<\/p>\n
